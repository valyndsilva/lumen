# Lumen — Claude Code Instructions

## Security
- NEVER read, cat, or display .env files or any file containing API keys/secrets
- NEVER log, print, or echo environment variables containing keys or tokens
- Only reference .env.example for environment variable names

## Project
- Use **pnpm** (not npm) for frontend package management
- Use **python3** (not python) for backend commands
- Backend virtual env: `backend/venv`

## Git
- Do NOT create commits — the user commits manually
- Do NOT push to remote unless explicitly asked
